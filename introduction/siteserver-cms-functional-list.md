# SiteServer CMS 功能清单

|功能名称|具体实现功能 
|-|-| 
|多服务器部署|1、支持父站点、子站点的创建与统一管理；<br>2、对每个站点设置不同的域名，一个站点内支持使用二级域名；<br>3、支持把不同的站点发布到不同的服务器上，将访问压力平均分配到多台服务器；<br>4、支持网站图片、文档、视频分别部署到不同的服务器上，以提高网站打开速度和安全性；<br>5、支持CMS系统与网站前台进行内外网分离部署从而大大提升系统的安全性；|
|内容多终端一体化|1、内容可以发布到包含PC、手机App（兼容iOS和Android平台）、微信、平板电脑等各种终端设备；<br>2、系统上内容共享互通，只需要发布一次，可能同时推送到PC和移动各端；<br>3、系统提供“PC+移动”一体化技术解决方案，可以为每个发布终端置不同的模板以最适合的形式展示；<br>4、系统还支持PC和移动各端统一一套响应式模板，从而在不同端上能显示相同的内容和不同的显示样式；|
|站点管理|1、支持无限个站点、子站点的创建，统一管理平台；<br>2、各站点展示以直观的树形导航形式展示；<br>3、各站点独立管理，互不影响，栏目、模板、管理员、网站配置等都独立管理；<br>4、站点间可以进行信息共享，内容可以在站内、站点间互相转移（手动和自动）；<br>5、每个站点配置独立域名；<br>6、支持将生成的网站发布到多台服务器上，便于负载均衡；<br>7、支持站点模板形式，利用建好的网站快速克隆一个新的网站；|
|管理员权限管理|1、系统可以添加多个管理员，管理员逐级管理；<br>2、系统支持角色自定义，可以赋予每个管理员不同管理角色，每个管理员可以拥有多个角色；<br>3、支持角色添加、修改与删除可以对某个管理角色赋予不用的权限；<br>4、理权限细致到具体的栏目、具体的功能；<br>5、总管理员可以为每个站点配置各自的站管理员；<br>6、站管理员还可以自己配置本站站内管理员，将权限下分；| 
|栏目（频道）管理|1、可根据需要自主创建栏目和设置栏目的分级；<br>2、无级数限制树结构栏目分级；<br>3、快速添加、删除栏目；<br>4、可以设置栏目组，方便不同的栏目组合显示，比如不同栏目组合形成某个专题；<br>5、可以对栏目在站内或者站与站之间进行批量转移，转移时可以将栏目下内容一并转移；<br>6、设置栏目索引，方便栏目调取；<br>7、多种栏目链接类型；<br>8、栏目可以设置外部链接；<br>9、栏目（包含本栏目内容）可以批量导出、导入，方便栏目的备份与还原；<br>10、栏目及栏目内容的访问路径可以自定义，方便栏目和栏目内容的存储；<br>11、对栏目页面关键字及页面的描述；| 
|内容管理|1、后台可以添加内容标题、作者、来源、摘要、发布时间、主题内容等；<br>2、后台预设多种内容添加字段，并可对字段进行添加、删除、修改等自定义工作；<br>3、集成多种类Word编辑器，文章发布支持文本、图片、附件、视频、flash等；<br>4、内容可以批量导出、导入，方便内容的备份与还原；<br>5、Word导入功能，可以在Word中编辑好内容后，直接导入到系统中；<br>6、可以设置内容组，方便不同的内容组合显示；<br>7、内容可以站内栏目间、站点间进行批量转移，方便内容的复用；<br>8、内容手动或自动分发，新增内容后，该篇内容自动在其他栏目出现，实现站内栏目之间、站点与站点间内容共享；<br>9、内容可以自由排序；<br>10、某篇内容可以设置外部链接；<br>11、多种形式的内容排序功能，可以按照多种形式进行内容组合和排列（置顶、醒目、推荐、热点等）；<br>12、内容发布支持多级审核机制，审核机制可以自由配置；<br>13、内容发布前预览功能；<br>14、内容图片水印、附件上传下载功能；<br>15、后台可对整站内容进行条件及关键字的搜索；<br>16、添加多种内容标签（Tag），内容可以按照不同标签进行分类调用显示，比如：相关新闻等；<br>17、对内容数目超过1万篇以上的栏目进行归档，减轻数据库压力；|
|内容审核|1、系统内嵌功能强大的工作流引擎，用于信息采、编、发的审核应用；<br>2、支持串联、并联审批工作流程；<br>3、五级审核机制，可以将每级审核机制赋予不同的管理员，同时可以配置给某个管理员多种审核权限；<br>4、对文章流转过程进行定义，包括草稿、最终审核通过等；<br>5、可以对审核不通过的进行逐级驳回，并附带驳回意见；|
|内容评论|1、提供对发布信息内容的评论功能，可以设置可否对某内容进行评论；<br>2、网友评论可以设置是否需要登录方可评论；<br>3、网友的评论只有被审核通过方可显示；|
|内容模型|1、可以对不同栏目类型定义不同内容模型，满足不同类型栏目的需求，如新闻栏目、图片栏目、互动交流、政务公开等；<br>2、添加内容，可以对内容的字段进行自定义，不同栏目可以设置不同的字段；<br>3、字段可以自由添加、修改、删除；<br>4、可以设置真实与虚拟字段两种类型；<br>5、可以设置联动字段；<br>6、可以添加其他多种类型的内容模型，以适应不用栏目需要；|
|内容回收站|1、删除的内容暂时存放在回收站中；<br>2、可以对回收站中的内容做彻底删除和恢复的操作；|
|站点数据统计|1、可以对每个管理员的工作量进行统计，对管理员添加的文章数进行统计；<br>2、可以对每个用户的投稿情况进行统计；<br>3、可以内容点击量进行统计；<br>4、可以对附件的下载量进行统计；|
|模板管理|1、提供模板自建功能，满足不同栏目风格的功能要求，支持页面风格变换及静态发布；<br>2、支持站点、栏目分级模板管理，便于维护，可方便浏览不同栏目下的模板；<br>3、可以通过模板的修改与匹配随意改变网站所有页面的显示样式；<br>4、提供完整的模板操作功能按钮，便于用户操作选择，包括模板增删改、备份、恢复；<br>5、支持不同栏目匹配不同模板，模板快速复制；<br>6、支持基于DreamWeaver的模板插件，进行模板可视化编辑；<br>7、支持多种模板格式，模板格式可定义为htm、 html 、shtml 、asp、jsp、xml等多种格式；<br>8、支持首页模板、栏目页模板、内容页模板、单页模板等多种形式；<br>9、制作模板无需编程，简单易学；|
|站点配置|1、支持直接修改站点名称；<br>2、支持静态和动态两种访问方式；<br>3、支持多语言编码，用于选择符合这个站点的编码规则；<br>4、支持自定义后台内容列表中默认每页显示的数目；<br>5、是否对内容点击量进行统计；|
|内容管理配置|1、支持内容组的开启和关闭；<br>2、支持标签的开启和关闭；<br>3、支持自动保存外部图片到本站，或者不保存；<br>4、支持内容是自动分页还是手动分页的选择，支持自定义分页每页字数；<br>5、设置内容审核，默认和多级别审核；<br>6、方便的支持后台栏目和内容列表的评论链接是否显示在后台；<br>7、支持图片内容模型预览图的自定义大小；|
|上传图片配置|1、支持上传文件夹的自定义；<br>2、支持不同的存入方式，按年、按年月、按年月日，存入各自目录；<br>3、支持上传文件是否自动改名；<br>4、上传图片类型、图片大小；上传媒体类型、媒体大小、等都可自定义配置；|
|图片水印配置|1、支持水印在图片上按九宫格显示；<br>2、支持水印透明度自定义；<br>3、水印分图片型和文字型；<br>4、图片型支持上传图片的大小自定义；|
|提交表单插件|1、可以新建多种表单形式，表单字段及显示样式可以自定义；<br>2、可以利用表单管理制作问卷调查、咨询等；<br>3、表单可以设置是否需要审核、是否回复；<br>4、提交的表单结果可以导出Excel；|
|投票插件|1、支持创建多个投票评选活动，每个投票评选可配置不同规则；<br>2、投票评选支持文本、图形、链接等多种形式；<br>3、投票评选支持是否启用验证码、是否登录后才能投票、是否允许同一IP（或同一注册用户）重复投票（允许的话是否有时间间隔）、投票结果是否允许在前台查看、投票的开始和结束时间、在投票前或后是否必须留下某些信息等相应参数的设置；<br>4、支持以图形和表格形式查看投票结果，包括投票汇总结果（把单选、多选子项的占比统计出来）以及投票明细；同时以excel形式导出投票汇总结果以及明细；|
|问卷调查插件|1、支持多个问卷调查，每个问卷调查可以配置不同规则；<br>2、问卷调查支持单选、多选、下拉、文本等多种形式；<br> 3、问卷调查支持是否登录后才能参与、是否允许同一IP（或同一注册用户）重复参与（允许的话是否有时间间隔）、问卷调查的开始和结束时间、是否必须留下某些信息（姓名、电话、学校等）等相应参数的设置；<br>4、支持以图形和表格形式查看问卷调查结果，包括问卷调查汇总结果（把单选、多选子项的占比统计出来）以及明细；同时可以以excel形式导出汇总结果以及明细；|
|信息采集插件|1、可以通过Web页面的链接规则和内容的页面采集Web内容，采集的内容可以设置是否审核后发布；<br>2、可以通过匹配数据库表里的字段来直接采集数据库里的数据；<br>3、可以采集某个网页上的所有数据如：CSS、图片、页面代码等，将该页面采集成Html静态页；|
|广告插件|1、可以管理网站上飘浮的广告位。广告分为：飘浮广告、全屏下推、弹出窗口；<br>2、可以制作网站上的固定广告位。广告位可以设置成：html、JS代码、文字、图片、flash；|
|搜索引擎优化插件|1、页面元数据，可以在页面的里加上Title、keyword、description等数据，可以让搜索引擎更容易的收录到网站；<br>2、站点地图，分为谷歌及百度，可以在根目录创建一个xml文件，让百度及谷歌更容易收录到网站；|
|流量统计插件|1、统计摘要，可以显示网站访问的基本信息；<br>2、站点流量统计，分为栏目流量统计、内容流量统计、内容流量排名。分别可以统计栏目的访问量、内容访问量及内容流量的排名；<br>3、分时统计，分为最近访问统计、24小时统计、天统计、月统计、年统计；<br>4、来源统计，分为受访页面统计、来访网址统计；<br>5、客户端统计，分为操作系统统计、来访网址统计；|
|站内链接插件|1、添加关键词和链接，可以让系统在生在内容页的时候，只要内容里有关键字就自己产生链接；<br>2、站内链接设置，设置站内链接生成的样式；|
|敏感词管理插件|1、可以逐个添加或批量导入敏感词库；<br>2、每个敏感词可以设置替换内容和敏感等级，不同等级会对应不同的操作；<br>3、内容发布时会提醒编辑人员内容中包含敏感词并高亮显示，编辑人员可以选择编辑或继续发布；<br>4、可以设置敏感词自动检测功能是否开启；|
|数据库备份恢复插件|1、数据备份，分为4个类型的备份。分别是显示模板、栏目及内容、文件、整站。用户可以根据自己的需要进行备份；<br>2、数据恢复，用户可以根据自己的需要来恢复数据；<br>3、管理备份文件，用户可以对以前进行备份的文件做删除操作；|
|日志管理插件|1、对使用系统所有用户的所有操作都提供完善的日志记载功能，当操作出现错误时，便于用户查看错误或警告产生的原因和出处；<br>2、可根据多条件来检索日志信息；<br>3、可以删除日志；|
|模板管理|1、提供模板自建功能，满足不同栏目风格的功能要求，支持页面风格变换及静态发布；<br>2、支持站点、栏目分级模板管理，便于维护，可方便浏览不同栏目下的模板；<br>3、可以通过模板的修改与匹配随意改变网站所有页面的显示样式；<br>4、提供完整的模板操作功能按钮，便于用户操作选择，包括模板增删改、备份、恢复；<br>5、支持不同栏目匹配不同模板，模板快速复制；<br>6、支持基于DreamWeaver的模板插件，进行模板可视化编辑；<br>7、支持多种模板格式，模板格式可定义为htm、 html 、shtml 、asp、jsp、xml等多种格式；<br>8、支持首页模板、栏目页模板、内容页模板、单页模板等多种形式；<br>9、制作模板无需编程，简单易学；|
|用户中心插件|1、系统支持网站会员注册与登录；<br>2、后台可以新添用户，管理员可以对会员进行搜索和管理；<br>3、管理员可以对注册会员群发邮件和短信；<br>4、会员在会员中心可以向管理员指定的网站栏目进行投稿，投稿审核后方可发布；<br>5、注册会员分组管理，不同组别会员具有不同的网站访问权限；|
