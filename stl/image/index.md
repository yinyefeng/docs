# stl:image 显示图片


### 定义

通过 stl:image 标签在模板中显示栏目或内容的图片

### 属性

属性  | 说明
------  | ------
channelIndex | 栏目索引
channelName | 栏目名称
no | 显示字段的顺序
parent | 显示父栏目
upLevel | 上级栏目的级别
topLevel | 从首页向下的栏目级别
type | 指定存储图片的字段
isOriginal | 如果是引用内容，是否获取所引用内容的值
src | 显示的图片地址
altSrc | 当指定的图片不存在时显示的图片地址
width | 宽度
height | 高度
